## Custom Badges

You can create custom badges using [shields.io](https://shields.io/):

```
https://img.shields.io/badge/<LABEL>-<MESSAGE>-<COLOR>
```

Colors: `brightgreen`, `green`, `yellow`, `yellowgreen`, `orange`, `red`, `blue`, `lightgrey`, `success`, `important`, `critical`, `informational`, `inactive`

Examples:
- ![Queries/sec](https://img.shields.io/badge/queries%2Fsec-24K-brightgreen)
- ![Inserts/sec](https://img.shields.io/badge/inserts%2Fsec-93K-brightgreen)
- ![Tests](https://img.shields.io/badge/tests-237-success)
